(pt_BR) Projeto inicialmente criado para fins de tese de conclusão de curso escrito com linguagens como: PHP, SQL, HTML, CSS, JAVASCRIPT... aprimorado para o mercado de aprendizado atual e suas necessidades. Tem por objetivo oferecer melhorias para a educação no Brasil como complemento a sala de aula. com vista à pandêmia no mundo, o impacto causado à educação poderia ser reduzido caso uma tecnologia como essa estivesse a livre acesso para escolas e alunos.



Próximos objetivos: Reformular todo o código e banco de dados do projeto, usando linguagens e frameworks como: Python, HTML, CSS, JavaScript, SQL, Django , VueJS, etc...
